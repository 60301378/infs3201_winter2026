# Assignment 4

## How to Run

1. Install dependencies:
npm install

2. Set MongoDB connection string (PowerShell):
$env:MONGO_URI="mongodb+srv://<username>:<password>@cluster0.4kmqmoh.mongodb.net"

3. Run the app:
node app.js

4. Open in browser:
http://localhost:8000
